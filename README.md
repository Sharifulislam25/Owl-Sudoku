## 🎮 How to Play

### **Play Online Instantly (Recommended)**

1. Go to your GitHub repository
2. Go to **Settings** → **Pages** (left sidebar)
3. Under **Source**, select **Deploy from a branch**
4. Select **main** branch and **/ (root)** folder
5. Click **Save**
6. Wait 2-3 minutes, then access at: `https://[your-username].github.io/[repo-name]/`

### **Download & Play Locally**

1. Click the **Code** button (green) above
2. Click **Download ZIP**
3. Extract the file
4. Open `sudoku.html` in any web browser

## ✨ Features

### 🎨 Design
- **Dark Mode** - Royal blue accents with futuristic RGB mechanical keyboard aesthetic
- **Light Mode** - Sky blue accents with clean minimalist design
- **Animated Welcome Screen** - Bouncing owl logo with smooth fade transitions
- **Futuristic Numpad** - Glowing LED-style keys with press animations

### 🎯 Gameplay
- **3 Difficulty Levels** - Easy, Medium, Hard
- **Smart Hints** - Get help when stuck
- **Solution Check** - Verify your progress
- **Timer** - Track your solving time
- **Save/Load** - Continue your game later

### 💾 Features
- **Local Storage** - Games auto-save
- **Statistics** - Track games played, won, and best time
- **Player Name** - Customize your profile
- **Theme Toggle** - Switch between dark/light modes instantly

### 🎬 Special Effects
- **Welcome Animation** - Animated owl with bouncing effect
- **Closing Animation** - Spinning numbers fading to black
- **Cell Highlighting** - Smart highlighting of related cells
- **Error Indicators** - Red highlighting for wrong answers

## 🦉 Why Owl Sudoku is Special

### 1. **Unique Owl Branding**
Every aspect of the game features a cute owl character - from the tab icon to the footer. The owl bounces during the welcome screen and says goodbye when you exit.

### 2. **Futuristic RGB Keyboard Design**
The number pad mimics a mechanical gaming keyboard with:
- Glowing LED effects
- 3D raised buttons
- Press animations
- Cyberpunk aesthetic

### 3. **Smooth User Experience**
- Seamless transitions between screens
- No page reloads during gameplay
- Instant theme switching
- Keyboard shortcuts (1-9 to fill, Backspace to erase, ESC to exit)

### 4. **Fully Offline**
No internet required after loading - works completely offline!

### 5. **Progress Persistence**
Your stats, saved games, and player name persist in local storage - even after closing the browser.

## 🕹️ Controls

| Input | Action |
|-------|--------|
| Click cell | Select cell |
| Click number | Fill selected cell |
| 1-9 keys | Fill selected cell |
| Backspace/Delete | Clear selected cell |
| ESC | Open exit confirmation |
| 🌓 button | Toggle theme |
